Description du projet

Le projet My Avatar est une application web inspirée du célèbre service Gravatar. Son objectif est de permettre à chaque utilisateur d’associer une photo de profil à son adresse e‑mail, accessible ensuite via une URL générée à partir du hash SHA256 de cette adresse.

L’application a été construite avec le framework Symfony et intègre divers modules comme Twig pour le rendu dynamique des pages, Doctrine pour la gestion des entités, et des requêtes AJAJ/JavaScript natif pour les interactions asynchrones (vérification du login, masquage de profil, etc.).

Ce projet a renforcé mes compétences en architecture MVC, sécurité web et gestion des rôles. Il m’a également sensibilisé à la séparation des responsabilités entre les couches backend et frontend dans une application Symfony.

Fonctionnalités principales

  • Inscription et authentification d’utilisateurs.
  • Ajout, modification et suppression de photo de profil.
  • Masquage/démasquage du profil en temps réel via AJAX.
  • Accès public aux avatars via URL sécurisée par hash SHA256.
  • Mode maintenance global configurable via fichier .env.
  • Gestion des rôles (utilisateur, administrateur).
  • Commandes Symfony pour la création et suppression d’utilisateurs depuis le terminal.
  • Interface d’administration listant les utilisateurs avec suppression sécurisée.
