Description du projet

J'ai effectué un stage à l'étranger chez Fekra Systems, où j'ai conçu et développé un système de gestion de rendez-vous pour une application médicale. Le back-end a été développé avec le framework Spring Boot, tandis que la partie front-end a été réalisée avec JavaFX. La base de données a été gérée avec PostgreSQL. Cette expérience m'a permis de renforcer mes compétences techniques et de découvrir le fonctionnement d'un environnement professionnel international.
