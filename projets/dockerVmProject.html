Description du projet

Application web permettant aux développeurs, étudiants et enseignants de composer, générer et déployer des stacks Docker sur machines virtuelles Proxmox via une interface graphique moderne.

L'utilisateur configure sa stack (images, ports, volumes, dépendances), génère un docker-compose.yml validé et un script deploy.sh, puis déploie sur VM. Les enseignants accèdent aux conteneurs étudiants pour suivi/correction.

Objectif : standardiser les environnements de dev (Windows/Linux/Mac), centraliser les fichiers, faciliter le suivi pédagogique et réduire le temps de mise en place des environnements de test.

Traces techniques

1. Accès au conteneurs de la VM

2. Accès SSH/SFTP sécurisé à la VM

Architecture technique

Chaque VM embarque un Docker Engine isolé. Les stacks sont déployées dans des conteneurs par utilisateur/groupe. WireGuard VPN pour accès réseau sécurisé.

Fonctionnalités principales

  • Interface drag & drop : composer stacks (images, ports, volumes, depends_on)
  • Bibliothèque stacks : LAMP, MERN, Symfony+PostgreSQL (clonables)
  • Sauvegarde/partage : configurations + versionning groupes
  • Authentification JWT : rôles étudiant/prof/admin
  • Déploiement VM : validation YAML + script deploy.sh auto
  • Accès services : IP:PORT web + SSH/SFTP + copie fichiers VM
  • Suivi pédagogique : profs accèdent conteneurs étudiants
