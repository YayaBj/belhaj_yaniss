Description du projet

Ce projet a pour objectif de développer une plateforme complète de critiques d’équipements hardware (composants informatiques, périphériques, etc.), basée sur une architecture client-serveur moderne.

Le backend est réalisé avec Symfony et exploite le framework API Platform afin de créer une API REST sécurisée conforme aux bonnes pratiques : gestion des rôles (utilisateur, administrateur), validation et publication des critiques, système d’authentification par JWT, ressources relationnelles et sous-ressources.

Le frontend en Vue.js consomme cette API via des appels asynchrones sécurisés. Il offre une interface utilisateur réactive et ergonomique pour consulter des fiches de matériel, publier des critiques, gérer ses favoris et voir ses propres contributions.

Le projet est actuellement en cours de développement. Les principales routes de l’API sont fonctionnelles, la base de données est opérationnelle, et les composants Vue.js sont en cours d’intégration pour une version utilisable publique.

Fonctionnalités principales

  • API REST Symfony avec API Platform (gestion CRUD sécurisée).
  • Système d’authentification JWT et gestion des rôles utilisateurs.
  • Création, modification et suppression d’objets critiquables (produits hardware).
  • Publication et validation des critiques (texte, note, critères).
  • Favoris utilisateurs et affichage des dernières critiques associées.
  • Consommation sécurisée de l’API via l’application Vue.js.
  • Intégration du service MyAvatar pour les photos de profil utilisateurs.
  • Gestion des erreurs, notifications et rafraîchissement dynamique via Vue.js.
